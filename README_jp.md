Roundはオープンソースの分散システム開発用のコンセンサスサービスフレームワークです。
Roundは「[円卓の騎士](http://ja.wikipedia.org/wiki/円卓の騎士)」が由来で,クラスターの全てのノードが等しい役割を持ち、クラスターにマスターノードは存在しないことを意味しています。

Round is architected for:

- Simplify
- Clarify
- ZeronConf
- Decentralized
- Autonomous
- Dynamics
